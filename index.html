XRP send widget

<script type="text/javascript" src="https://raw.githubusercontent.com/kikkikkikkik/xrp-send-widget/gh-pages/xrp-send-widget.js"></script>
 
<script>
	xrp_send_widget.set({
		to : "rfSSHTYGrQDZigrvvXyia2j6PCvAZADDD1",
		amount : "10",
		dt : "265",
		text : "Send XRP (open your wallet)",
		show_count : true,
		show_amount : true,
		show_title : true,
	});
</script>
Parameter Description
to Your wallet address. (required)
amount Show default amount.
dt The number to identify transactions. Default value is 265.
text Change the text on button. Default value is "Tip XRP".
show_count Set true to show the total number of transactions.
show_amount Set true to show the total amount of XRP received.
show_title Set true if you want to show your wallet address when the mouse pointer is over the button.
